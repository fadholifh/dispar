Categories widget! Display lists of categories in website sidebar.

== Description ==

The "WP Categories Widget" is a very simple plugin to display the list of categories als there are given an option to choose type of taxonomy to display categories of any taxonomy type.

Note : please don't forget to leave your valuable feedback and suggestions.

  **Send your query to [AUTHOR](mailto:raghunath.0087@gmail.com)**


  Video Tutorial : 
  
  https://youtu.be/pbsnErpu_8U
  

= `Features` = 
 * An option to show categories post count
 * An option to hide widget title
 * An option to show categories of any taxonomies type
 * An option to exclude/include categories of the taxonomies
 
 
== Frequently Asked Questions ==

1.) Can I hide widget title?
csv
 * Yes, You can hide widget title if you do not show in sidebar.
 
2.) Can I show list of category from my custom taxonomy?

 * Yes,You can choose your custom taxonomy for show list of their all categories.

3.) Can I exclude specific category from list?

 * Yes, you can exclude catgories from list.

== Installation ==

Installing the plugin:

1.  Unzip the plugin's directory into `wp-content/plugins`.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  The categories post will be available on widegt page (Apperance >> Widget)
    WordPress administration page.
